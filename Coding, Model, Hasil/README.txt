# Coding, Model, Hasil

**Deskripsi**:  
Folder ini berisi berbagai kode program, model, dan hasil eksperimen yang digunakan dalam penelitian tesis. Setiap folder mewakili model yang berbeda, sementara file `#1-Data_Understanding.ipynb` berisi pemahaman awal terkait data yang digunakan dalam eksperimen dan  pre-processing data yang akan menghasilkan file `Prepocessed (3).xlsx`

**Isi Folder**:
1. **SVM**: Folder ini berisi kode dan hasil eksperimen menggunakan model klasifikasi SVM (Support Vector Machine) untuk analisis.
2. **RF**: Folder ini berisi kode dan hasil eksperimen menggunakan model klasifikasi Random Forest untuk analisis.
3. **CNN**: Folder ini berisi kode dan hasil eksperimen menggunakan model klasifikasi Convolutional Neural Network (CNN).
4. **BiLSTM**: Folder ini berisi kode dan hasil eksperimen menggunakan model klasifikasi BiLSTM (Bidirectional Long Short Term Memory).
5. **4 Terbaik (70-30)**: Folder ini berisi file yang menunjukkan perbandingan antara empat model terbaik, dengan pembagian data 70% untuk pelatihan dan 30% untuk pengujian.

**File Utama**:
- **#1-Data_Understanding.ipynb**: Notebook ini memberikan pemahaman dasar tentang dataset yang digunakan dalam eksperimen dan langkah-langkah pra-pemrosesan yang dilakukan.

**Panduan Penggunaan**:
1. **SVM, RF, CNN, BiLSTM**: Setiap folder berisi file kode yang dapat dijalankan dengan menggunakan Jupyter Notebook atau IDE lainnya yang mendukung Python.
2. Jalankan file **#1-Data_Understanding.ipynb** untuk memulai dengan pemahaman data dan eksplorasi lebih lanjut sebelum melanjutkan ke eksperimen model.
3. Hasil dari masing-masing model dapat ditemukan dalam subfolder masing-masing.

**Prasyarat**:
- Python 3.x
- Library yang dibutuhkan: `scikit-learn`, `tensorflow`, `keras`, `pandas`, `numpy`, `matplotlib`, dan library lain yang sesuai dengan eksperimen.

**Lisensi**:  
Dataset dan kode ini tersedia untuk penggunaan penelitian akademik. Jika digunakan untuk tujuan komersial, izin tambahan harus diperoleh.
